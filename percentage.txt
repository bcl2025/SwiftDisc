SwiftDisc Feature & Phase Completion (as of 0.5.1)
===================================================

Phases
------
- Phase 1: Gateway Stability ............ 100%
  Notes: Identify/Resume/Reconnect; heartbeat ACK recovery; intents; READY/MESSAGE_CREATE/GUILD_CREATE/INTERACTION_CREATE; presence helper; shard field.

- Phase 2: REST Maturity ................. 100%
  Notes: Per-route limits + retries; detailed error decoding; endpoints: Channels, Guilds, Interactions, Webhooks.

- Phase 3: High-Level API ................ 100%
  Notes: AsyncSequence-first; callback adapters; prefix command framework; initial caching (recent messages); helper utilities.

- Phase 4: Cross-Platform Excellence ..... 90%
  Notes: URLSession WebSocket adapter used across platforms; CI added (macOS+Windows); embeds + slash commands complete; platform optimizations ongoing.

------------------
- Embeds ................................ 100%

- Slash Commands ........................ 85%

- Windows WebSocket Adapter ............. 100%

- Testing & CI .......................... 50%
